Sobre Mim

Minha jornada na ciência de dados

Minha História

Sou Eric Alves, estudante de Estatística na Anhembi Morumbi e apaixonado por ciência de dados. Atualmente trabalho como motorista de aplicativo, uma experiência que me ensinou muito sobre análise de padrões, otimização de rotas e tomada de decisões baseadas em dados em tempo real.

Esta experiência prática me motivou a investir na minha formação em Ciência de Dados através de cursos especializados, como o curso de Cientista de Dados pela Comunidade DS e Power BI pela DIO.

O que me motiva

O que mais me entusiasma na ciência de dados é a capacidade de encontrar histórias escondidas nos números e transformar dados brutos em insights acionáveis que podem impactar positivamente os negócios e a sociedade.

Meu Objetivo

Estou buscando uma oportunidade para trabalhar profissionalmente como Cientista de Dados, onde posso aplicar meus conhecimentos para melhorar a tomada de decisão empresarial através da construção de soluções inovadoras usando dados.

Projetos em Destaque

Soluções práticas em ciência de dados

Dashboard Interativo Zomato

Dashboard interativo desenvolvido com Streamlit para análise de dados de restaurantes globais, explorando métricas como custos médios por país, tendências culinárias e avaliações.

Python Streamlit Pandas Plotly

Principais Resultados:

  • Identificação de padrões de preços por região
  • Análise de popularidade de culinárias
  • Dashboard responsivo e intuitivo

Análise Preditiva de Demanda de Transporte

Modelo de machine learning para prever demanda de transporte por aplicativo, utilizando dados históricos e fatores externos como clima e eventos.

Python Scikit-learn Pandas Matplotlib

Principais Resultados:

  • Precisão de 85% na previsão de demanda
  • Identificação de padrões sazonais
  • Otimização de recursos operacionais

Análise de Sentimento de Reviews

Sistema de processamento de linguagem natural para análise de sentimento em avaliações de produtos e serviços, classificando opiniões como positivas, negativas ou neutras.

Python NLTK TextBlob WordCloud

Principais Resultados:

  • Classificação automática de sentimentos
  • Visualização de palavras-chave
  • Insights para melhoria de produtos
